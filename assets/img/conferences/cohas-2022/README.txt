Drop conference photos here. Filenames are sorted alphabetically — prefix with
01_, 02_, etc. if you want a specific order.

Recommended:
  - JPG, PNG, WEBP, or AVIF
  - 1000–1500 px wide
  - Under 500 KB each (resize before adding if larger)

After adding photos, run from the repo root:
  python3 build_site.py

Then commit and push. The conferences page rebuilds with your photos in a
right-to-left sliding gallery for this entry.
